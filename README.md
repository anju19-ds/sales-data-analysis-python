# sales-data-analysis-python
Sales Data Analysis using Python, Pandas and Matplotlib

## Project Overview
This project analyzes a retail sales dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Total Sales and Profit Calculation
- Region-wise Sales Analysis
- Category-wise Profit Analysis
- Top Performing Sub-Categories

## Insights
- Identified highest performing regions
- Analyzed profitable product categories
- Visualized business trends using charts
